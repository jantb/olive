# olive
Editor experiment in Go.
Forked from linde12/kod.

# Getting started
Make sure "xi-core" is in your path
```
> clone https://github.com/xi-editor/xi-editor
> cd xi-editor
> cd rust
> cargo install --force
> cd syntect-plugin
> make install
```
Then install olive
```
> go get github.com/jantb/olive`
> olive filename.go
```

# Shortcuts:
CTRL-S Save file
CTRL-C Quit
